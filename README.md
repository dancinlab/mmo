# mmo — game-art image prompts

Image-generation prompts for game / MMO marketplace thumbnails. Each prompt targets an image
model (`fal-ai/gpt-image-2`), 16:9, no baked price/currency.

# 🧊 Block (voxel) dioramas (32)

Game/MMO scenes built out of **smooth 3D blocks of mixed, non-uniform sizes** (big blocks for masses + many small blocks for detail, finely-subdivided characters, open-air outdoor scenes with cinematic lighting, no studs/grooves),
isometric tilt-shift miniature on a glossy pedestal, palette-varied.
Prompts: [`prompts-block.md`](./prompts-block.md) · scenes: [`scenes.json`](./scenes.json) · generator: [`gen-dioramas.py`](./gen-dioramas.py).
